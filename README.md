Dns Server written entirely in rust.

HOW TO USE 
1. clone the github repo and run the code in the terminal.
2. open a seperate terminal and use "dig "@127.0.0.1" -p 6969 <website name>"
   [example: dig "@127.0.0.1" -p 6969 www.google.com]
3. open the terminal where the code is running and you should have your results.
